`docs/features/active/2026-07-02-exchange-rbac-scripts-111/runbooks/exchange-rbac-setup.runbook.md` is this repository's first human-exception runbook (F11, exchange-rbac-scripts, issue #111) and is the structural precedent later features are asked to match. It uses a short header block (Feature / Runbook path / Authored / Source checklist) above the five skill-required sections (Cue, Prerequisites, Step-by-step Instructions, Verification, Source and Citation), numbers each instruction step, and ends with a Source-and-Citation table mapping each step to a dated Microsoft Learn URL.

**How to apply:** When asked to author a new `human-exception-runbook`, read this file first for the expected shape (header block + five sections + per-step citation table) before drafting. See also [[project-no-mcp-doc-tool-web-second-only]] for the sourcing-mechanism constraint that applies to the Source and Citation section.
