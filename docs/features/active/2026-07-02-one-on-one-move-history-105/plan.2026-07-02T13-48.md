### Phase 2 — Pure OneOnOneMoveGuard and SeriesMoveHistoryAnswers

- [x] [P2-T1] Create `src/OpenClaw.Core/Agent/OneOnOneMoveGuard.cs` (namespace `OpenClaw.Core.Agent`, pure static class, no I/O, no clock, no store reference) containing: (1) `public sealed record SeriesMoveHistoryAnswers(int MovesInLastSixOccurrences, bool MovedPreviousWeek);` (2) `public static SeriesMoveHistoryAnswers ComputeAnswers(IReadOnlyList<DateTimeOffset> movedOccurrenceStartsUtc, IReadOnlyList<DateTimeOffset> occurrenceStartsUtc, DateTimeOffset candidateOccurrenceStartUtc)` implementing D1/D2 exactly — anchor `Anchor(t) = t.UtcDateTime.Date`; window = the six greatest distinct anchors in `{Anchor(o) : o in occurrenceStartsUtc} ∪ {candidateAnchor}` that are `<= candidateAnchor` (all of them when fewer than six exist); `MovesInLastSixOccurrences` = count of entries in `movedOccurrenceStartsUtc` (per move row, not per distinct anchor) whose anchor is a member of the window set; `MovedPreviousWeek` = true iff any moved anchor `a` satisfies `candidateAnchor - 7 days <= a < candidateAnchor` (half-open); throws `ArgumentNullException` for null list arguments; XML docs document the caller-supplied occurrence-list input contract (D2: incomplete lists are conservative — the guard blocks more); (3) `public static bool CanMove(NormalizedMeetingContext meeting, string ownerEmail, string requesterEmail, OwnerPriority requestPriority, OwnerSchedulingPolicy policy, SeriesMoveHistoryAnswers history)` — throws `ArgumentNullException` for null reference arguments matching `MovePolicy.CanMove`; for every other kind returns `MovePolicy.CanMove(meeting, ownerEmail, requesterEmail, requestPriority, policy)` with the same arguments, unchanged; (4) `public static string ResolveSeriesKey(NormalizedMeetingContext meeting)` — throws `ArgumentNullException` for a null meeting; returns `SeriesMasterId` when non-null/non-empty, else `EventId` when non-null/non-empty, else throws `ArgumentException` (D3); `MovePolicy.cs`, `RecurringMeetingClassifier.cs`, and `NormalizedMeetingContext.cs` are not modified

- [x] [P2-T2] Create `tests/OpenClaw.Core.Tests/Agent/OneOnOneMoveGuardTests.cs` (MSTest + FluentAssertions) covering: (a) `CanMove` truth table for a `ONE_ON_ONE` context over `MovesInLastSixOccurrences` in {0, 1, 2} crossed with `MovedPreviousWeek` in {true, false} — allowed only for (0,false) and (1,false); (b) `ComputeAnswers` window boundary at exactly six occurrences — a move anchored on the seventh-most-recent occurrence is excluded from the count while a move anchored inside the six-anchor window is counted; (c) fewer-than-six known occurrences — all supplied anchors form the window; (d) previous-week boundaries — moved anchor at exactly candidate −7 days yields `MovedPreviousWeek == true`, −8 days yields false, −1 day yields true, and a moved anchor equal to the candidate's own anchor yields false; (e) same-UTC-day double move — two moved entries with distinct starts on the same anchor date count as 2 (per move row, window membership per anchor); (f) occurrence anchors strictly greater than the candidate anchor are excluded from the window; (g) `ResolveSeriesKey` returns `SeriesMasterId` when present, falls back to `EventId` when `SeriesMasterId` is null or empty, and throws `ArgumentException` when both are null/empty; (h) `ArgumentNullException` guards — null lists for `ComputeAnswers`, null `meeting`/`ownerEmail`/`requesterEmail`/`policy`/`history` for `CanMove`, null `meeting` for `ResolveSeriesKey`
